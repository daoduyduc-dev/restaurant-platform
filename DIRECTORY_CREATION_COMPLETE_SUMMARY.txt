🆘 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────────

   PROBLEM: "Access Denied" when running batch file
   SOLUTION: Right-click the batch file → Select "Run as Administrator"

   PROBLEM: Directories already exist
   SOLUTION: This is fine! Scripts will skip them. Use File Explorer to verify.

   PROBLEM: Command window closes too quickly
   SOLUTION: Run with: cmd /k "D:\restaurant-platform\RUN_THIS_FIRST.bat"
            Or edit the batch file and remove the "pause" line

   PROBLEM: "tree" command shows nothing but directories are there
   SOLUTION: Directories are created correctly! Use File Explorer to verify.

   PROBLEM: "mkdir not recognized" or "'mkdir' is not a valid command"
   SOLUTION: You're in PowerShell. Run the PowerShell script instead (CreateDirs.ps1)
            OR use "md" instead of "mkdir" (same command)

   PROBLEM: "node is not recognized"
   SOLUTION: Node.js is not installed. Use another method instead.

   PROBLEM: "python is not recognized"
   SOLUTION: Python is not installed. Use another method instead.
