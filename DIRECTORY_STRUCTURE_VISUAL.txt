🎓 LEARNING NOTES:
─────────────────────────────────────────────────────────────────────────────────

    This follows Maven standard directory structure:
    • src/main/java - Production code
    • src/test/java - Test code (what we're creating)
    • src/test/resources - Test resources and configuration

    Package structure: com.restaurant.platform.modules.*
    • Follows Java naming conventions
    • Allows proper test discovery by Maven
    • Enables IDE integration for test running

    Test modules mirror production modules:
    • Production: src/main/java/.../ modules/*
    • Tests:      src/test/java/.../ modules/* ← (6 directories)
    • Resources:  src/test/resources/ ← (1 directory)
