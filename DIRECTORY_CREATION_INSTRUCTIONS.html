✓ Option 3: Manual Command Prompt

Steps:

  1. Press Win + R
  2. Type: cmd and press Enter
  3. Copy and paste the following commands:

    mkdir src\test\resources
    mkdir src\test\java\com\restaurant\platform\modules\auth
    mkdir src\test\java\com\restaurant\platform\modules\menu

🆘 Troubleshooting

If directories already exist: The scripts use the mkdir command which will not fail if directories already exist. They will simply be skipped.
If you get "Access Denied" error:
  1. Right-click the batch file and select "Run as Administrator"
  2. Or open Command Prompt as Administrator first, then run commands
If "tree" command is not found: Use File Explorer to verify the directories were created, or use: dir /S src\test instead.
