🆘 TROUBLESHOOTING

ISSUE: "The system cannot find the path specified"
SOLUTION: Make sure you're in the correct directory
          Run: cd /d D:\restaurant-platform (with /d flag for drive change)

ISSUE: "mkdir is not recognized as an internal or external command"
SOLUTION: You're in PowerShell. Either:
          • Use CreateDirs.ps1 instead, OR
          • Use "md" instead of "mkdir" (same thing)

ISSUE: "tree" command doesn't work
SOLUTION: The directories are created correctly.
          Use File Explorer to verify instead.

ISSUE: The batch window closes immediately
SOLUTION: The execution is complete! Directories were created.
          The window closes normally after the pause command.

ISSUE: "Access Denied" error
SOLUTION: Right-click the script and select "Run as Administrator"

ISSUE: Nothing happens when I double-click the batch file
SOLUTION: Try one of these:
          1. Right-click and "Run as Administrator"
          2. Open Command Prompt and run: D:\restaurant-platform\RUN_THIS_FIRST.bat
          3. Use the Python or Node.js alternative

ISSUE: I see error messages but directories are created
SOLUTION: This is normal if directories already existed.
          The error messages are suppressed but some may show.
          The directories are still created correctly.
